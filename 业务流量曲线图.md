**基本信息：**

请求类型：HTTP

接口地址：

请求方式：GET

数据类型：X-WWW-FORM-URLENCODED

响应类型：JSON

接口状态：未启用

**请求参数：**

| **字段** | **类型** | **默认值** | **是否必须** | **说明** |
| --- | --- | --- | --- | --- |
| starttime | long |  | 否 | 查询开始时间 |
| endtime | long |  | 否 | 查询结束时间 |

**响应数据：**

| **字段** | **类型** | **说明** | **备注** |
| --- | --- | --- | --- |
| status | int | 状态 |  |
| msg | string | 返回信息 |  |
| data | jsonArray | 返回数据 | 表1 |

**表1：**

| **字段** | **类型** | **说明** | **备注** |
| --- | --- | --- | --- |
| unit | string | 流量单位 |  |
| list | jsonArray | 流量列表 | 表2 |

**表2：**

| **字段** | **类型** | **说明** | **备注** |
| --- | --- | --- | --- |
| utc | date | 时间 | new Date\(2016,11,01,09,01\) |
| flow | float | 流量数值 |  |
